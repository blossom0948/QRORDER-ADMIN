# QRORDER Admin

사장님용 QR 테이블 주문 관리 사이트입니다.

## 배포 URL

https://blossom0948.github.io/QRORDER-ADMIN/

## 역할

- 주문 대기, 조리중, 완료 상태 관리
- POS형 운영 대시보드 UI
- 새 주문과 직원 호출 접수 시 딩동 알림
- 전체, 음식 주문, 직원 호출 필터
- 진행 테이블, 직원 호출, 품절 메뉴, 최장 대기 시간 운영 요약
- 실시간, QR 보안, 메뉴 사진, 결제 연동 준비 상태 표시
- Firebase/Supabase, PG, QR 토큰 운영 연동 안내 패널
- 주문 요청사항 확인
- 음식 주문과 직원 호출 요청 구분 표시
- 메뉴 이미지 URL 설정
- 메뉴 이미지 파일 업로드 및 즉시 미리보기
- 메뉴명, 설명, 분류, 배지, 옵션 그룹 수정
- 메뉴 노출 순서 변경
- 메뉴 가격 설정
- 품절 여부 설정
- 메뉴 추가 및 삭제

손님용 주문 화면은 별도 저장소 `QRORDER-CUSTOMER`에서 배포합니다.
